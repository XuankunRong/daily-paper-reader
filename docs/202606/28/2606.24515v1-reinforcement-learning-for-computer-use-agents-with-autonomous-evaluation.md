## 摘要
计算机使用智能体（CUAs）通过直接在图形用户界面中感知和行动来执行高级用户目标。然而，面向CUAs的强化学习仍然困难，因为开放的桌面环境很少提供可扩展的、机器可读的奖励信号：任务成功通常依赖于视觉判断，难以通过手工设计的奖励函数或密集的人工标注来指定。我们提出了一种强化学习微调框架，该框架利用自主视觉-语言评估作为GUI智能体的可扩展监督信号。给定最终截图和原始指令，视觉-语言模型判断任务完成情况并提供终止反馈，在策略优化过程中无需任务特定的启发式规则或人工标注。由于自主评估器不完美，我们将其反馈建模为带噪的二元奖励通道，并推导出用于近端策略优化的噪声校正奖励估计器。在macOSWorld、Windows Agent Arena和OSWorld上的实验表明，校正后的评估器奖励优于零样本基线和原始评估器奖励，相较于零样本性能平均提高了12.6个百分点，相较于原始评估器微调提高了5.1个百分点。这些结果表明，当评估器噪声被显式建模和校正时，自主评估可以作为GUI环境中强化学习的实用奖励信号。

## Abstract
Computer-Use Agents (CUAs) execute high-level user goals by perceiving and acting directly within graphical user interfaces. However, reinforcement learning for CUAs remains difficult because open-ended desktop environments rarely provide scalable, machine-readable reward signals: task success is often visually grounded and hard to specify with handcrafted reward functions or dense manual labels.   We propose an RL fine-tuning framework that uses autonomous vision-language evaluation as a scalable supervision signal for GUI agents. Given a final screenshot and the original instruction, a Vision-Language Model judges task completion and provides terminal feedback without task-specific heuristics or manual labels during policy optimization.   Because autonomous evaluators are imperfect, we model their feedback as a noisy binary reward channel and derive a noise-corrected reward estimator for Proximal Policy Optimization. Experiments across macOSWorld, Windows Agent Arena, and OSWorld show that corrected evaluator rewards outperform both zero-shot baselines and raw evaluator rewards, improving success rates by an average of 12.6 percentage points over zero-shot performance and 5.1 points over raw evaluator fine-tuning. These results suggest that autonomous evaluation can serve as a practical reward signal for RL in GUI environments when evaluator noise is explicitly modeled and corrected.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
将 VLM 评估器输出的二元任务完成判断（成功/失败）视为一个带噪的二元奖励通道，显式建模其假阳性（false positive）和假阴性（false negative）错误率，并推导出一个渐近无偏的校正奖励估计器，直接嵌入到近端策略优化（PPO）中。

### 关键技术细节
- **评估器**：使用 Qwen2-VL-7B 模型，零样本地根据最终截图和原始指令输出二元信号 $\tilde{r}_i \in \{0,1\}$（1 表示任务完成）。
- **噪声模型**：定义假阴性率 $e^+ = \Pr(\tilde{r}=0 \mid r^\star=1)$，假阳性率 $e^- = \Pr(\tilde{r}=1 \mid r^\star=0)$，以及可分离常数 $D = 1 - e^+ - e^- > 0$（保证评估器优于随机猜测）。
- **校正估计器**：利用条件期望推导出 $\tilde{r}$ 的无偏表达式，得到校正奖励 $\hat{r}_i = (\tilde{r}_i - e^-)/D$，满足 $\mathbb{E}[\hat{r}_i \mid \tau_i] = r^\star_i$。
- **实际使用**：在独立于 RL 训练的校准集上估计 $\hat{e}^+$、$\hat{e}^-$ 和 $\hat{D}$，得到插值估计器 $\hat{r}^\sharp_i = (\tilde{r}_i - \hat{e}^-)/\hat{D}$，用于 PPO 的终末奖励。
- **策略优化**：奖励为终末形式，折扣回报 $\hat{R}_i = \gamma^T \hat{r}_i$，使用 PPO 优化目标 $J(\theta) = \mathbb{E}[\hat{R}_i]$，优势估计通过将校正奖励反向传播到轨迹中计算。

1. **自主评估可以作为一种可扩展的 RL 奖励信号**，但需要显式建模和校正评估器噪声。

## 8. 不足与局限

1. **评估器非平稳性**：假设错误率固定，但训练中智能体访问的状态分布会变化，可能导致噪声特性漂移，引入残余偏差。
2. **奖励稀疏性**：仅依赖终末二元信号，未引入中间奖励或进度估计，探索效率可能受限，尤其对长周期任务。
3. **校准集依赖性**：校正效果取决于校准集质量和大小；若 $D$ 很小（评估器接近随机），校正会放大噪声，训练可能不稳定。
4. **校正奖励可能超出 [0,1] 范围**，虽理论无偏，但实际 PPO 优化需注意归一化和超参数调整。
5. **评估器盲点**：VLM 可能基于表面视觉线索误判，忽略语义正确的中间状态或多解路径。
6. **实验覆盖不足**：未测试多种基座模型、多种评估器、更多种奖励校正策略（如动态估计、置信度加权），也未评估对分布外场景（新应用版本、不同分辨率）的鲁棒性。
