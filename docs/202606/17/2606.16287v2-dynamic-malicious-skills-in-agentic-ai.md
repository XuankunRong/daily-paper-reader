## 摘要
技能是智能体AI的关键使能组件。虽然它们增强了智能体的能力，但也引入了新的攻击面。在这项工作中，我们通过展示动态恶意技能来研究这种攻击面。通过将恶意指令嵌入自然语言文档（例如SKILL.md），攻击者可以诱导智能体在执行过程中将恶意逻辑动态注入原本良性的技能中。我们在OpenHands和Claude Code等智能体框架上评估了这种攻击，表明动态恶意技能能够在运行时以相当的成功率引入一系列恶意行为。为了缓解这一漏洞，我们提出了一种系统级防御措施，通过操作系统内核强制的只读挂载来阻止技能的动态修改。我们的评估表明，这种防御在保持良性技能功能的同时，有效地阻止了动态恶意技能。

## Abstract
Skills are a key enabling component of agentic AI. While they enhance agents' capabilities, they also introduce new attack surfaces. In this work, we investigate one such attack surface by demonstrating dynamic malicious skills. By embedding malicious instructions in natural-language documentation (e.g., SKILL.md), an attacker can induce an agent to dynamically inject malicious logic into an otherwise benign skill during execution. We evaluate this attack across agentic frameworks such as OpenHands and Claude Code, showing that dynamic malicious skills can successfully introduce a range of malicious behaviors at runtime with non-trivial success rates. To mitigate this vulnerability, we propose a system-level defense that prevents dynamic modification of skills using operating system kernel-enforced read-only mounts. Our evaluation demonstrates that this defense effectively blocks dynamic malicious skills while preserving the functionality of benign skills.

## 6. 论文的主要结论与发现

1. **动态恶意技能威胁严重**：在多种LLM上平均ASR达18.7%~43.7%，其中响应时间放大（最高82%）和本地文件删除具有普遍高危性。
2. **模型能力提升增加脆弱性**：Qwen3-8B到Qwen3.6-35B，ASR从18.7%升至41.6%；更强的指令遵循能力使攻击更易成功。
3. **安全对齐并非万能**：GPT-5和Sonnet 4.6虽然在某些行为上更稳健，但在其他行为上仍脆弱（如Sonnet 4.6响应时间放大ASR 82%）。
4. **提示级防御效果有限**：系统提示防御仅降低ASR约20.2%，且不同行为下降幅度差异大。
5. **权限级防御有效**：内核级只读挂载+拷贝监控器实现0.0% ASR，且不影响良性技能功能（300个良性技能全部成功完成任务）。
6. **现有注入检测方法不适用**：PromptArmor漏报高（FNR 85.8%），DataSentinel误报高（FPR 56.0%）。

## 7. 优点

- **攻击新颖**：首次提出动态恶意技能，揭示代理框架中运行时修改技能的脆弱性。
- **指令设计精巧**：通过先决条件框架、入口脚本锚定、包装突破三个子属性，使恶意指令高度逼真，难以被拒。
- **防御方案实用**：基于观察“良性技能极少修改代码”提出的最小权限防御，实际有效且不影响正常功能。
- **评估全面**：多个LLM、多种行为、消融实验、对比多种防御和检测方法，结论可靠。
- **数据集真实**：使用真实技能市场Skillsmp的15000+技能，有代表性。

## 8. 不足与局限

- **实验覆盖有限**：专有LLM（GPT-5等）仅用50个技能，可能引入偏差；未测试更多框架（如LangChain, AutoGPT）和更多变体技能（非Python技能）。
- **攻击假设较强**：攻击者需事先了解技能代码结构以生成有效指令；用户使用的框架、系统提示等未知但实验显示仍有效，但真实环境可能存在差异。
