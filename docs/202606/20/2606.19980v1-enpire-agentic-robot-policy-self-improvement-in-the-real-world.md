## 摘要
在真实世界中实现灵巧的机器人操作在很大程度上依赖于人工监督和算法工程，这成为追求通用物理智能的核心瓶颈。尽管新兴的编码智能体能够生成代码以自动化算法搜索，但其成功仍主要局限于数字环境。我们推测，自动化机器人研究缺少的是一个用于真实世界策略改进的可重复反馈循环：重置场景、执行策略、验证结果并优化下一次迭代。为弥合这一差距，我们引入了ENPIRE，这是一个为编码智能体设计的框架，通过四个核心模块实例化这种物理反馈例程：环境模块（EN）用于自动重置和验证，策略改进模块（PI）启动策略优化，部署模块（R）用于评估一个或多个并行运行的物理机器人的策略，以及进化模块（E），其中编码智能体分析日志、查阅文献、改进训练基础设施和算法代码以解决失败模式。这种闭环系统将真实世界的操作学习转化为可控的优化过程，最小化人工努力，同时允许在训练方案和智能体变体之间进行公平的消融实验。借助ENPIRE，前沿的编码智能体可以自主训练策略，在具有挑战性的灵巧操作任务上实现99%的成功率，例如整理针盒、系紧扎带和工具使用，当我们派遣智能体团队在机器人集群上工作时，这一过程会进一步加速。我们的结果表明了一种实用且可扩展的路径，即部署编码智能体在物理世界中自主推进机器人技术。

## Abstract
Achieving dexterous robotic manipulation in the real world heavily relies on human supervision and algorithm engineering, which becomes a central bottleneck in the pursuit of general physical intelligence. Although emerging coding agents can generate code to automate algorithm search, their successes remain largely confined in digital environments. We conjecture that the missing abstraction to automate robotics research is a repeatable feedback loop for real-world policy improvement: reset the scene, execute a policy, verify the outcome, and refine the next iteration. To bridge this gap, we introduce ENPIRE, a harness framework for coding agents that instantiates this physical feedback routine with four core modules: an Environment module (EN) for automatic reset and verification, a Policy Improvement module (PI) that launches policy refinement, a Rollout module (R) to evaluate policies with one or multiple physical robots operating in parallel, and an Evolution module (E) in which coding agents analyze logs, consult literature, improve training infrastructure and algorithm code to address failure modes. This closed-loop system transforms real-world manipulation learning into a controllable optimization procedure, minimizing human effort while allowing fair ablations across training recipe and agent variants. Powered by ENPIRE, frontier coding agents can autonomously train a policy to achieve a 99% success rate on challenging, dexterous manipulation tasks, such as organizing a pin box, fastening a zip tie, and tool use, a process that further accelerates when we dispatch an agent team on a robot fleet. Our results suggest a practical and scalable path toward deploying coding agents to autonomously advancing robotics in the physical world.

## 2. 方法论
### 核心思想
- 将物理操作学习分解为两个阶段：
  1. **环境构建阶段（EN）**：由人类提供一次性的反馈指导，编码智能体利用工具调用构建自动重置、安全约束、自动验证的环境API，并离线优化。
  2. **自动策略改进阶段（PIRE）**：编码智能体基于环境自动验证信号，独立探索优化方法（如行为克隆、强化学习、启发式学习等），迭代提升真实世界任务成功率。
- 四个核心模块：
  - **环境模块（EN）**：硬安全约束、自动验证（合成二元奖励函数）、自动重置（通过感知与规划API）。
  - **策略改进模块（PI）**：允许智能体查阅文献、提出假设、直接修改训练代码。
  - **部署模块（R）**：在单台或多台物理机器人上并行执行策略，收集轨迹与日志。
  - **进化模块（E）**：多智能体通过Git异步协作，共享/合并成功配方，加速假设筛选。

### 关键技术细节
- 自动验证奖励通过组合视觉（SAM3、BundleSDF）、力觉（扭矩信号）等感知方式，由智能体自动设计，延迟<150ms。
- 自动重置利用CuRobo规划、SAM3定位等工具，实现稳健的长时序复位。
- 多智能体协作：每个机器人与一个智能体绑定，通过Git分支共享代码和结果，智能体可自主合并或采纳同伴的改进。
- 策略学习支持多种范式：启发式学习、行为克隆（BC）、离线/在线强化学习、离线到在线RL、VLA（视觉-语言-动作模型）与程序代码结合等。

## 6. 主要结论与发现
- ENPIRE成功实现了几乎所有评估任务的高成功率（Push-T、Pin Insertion达99%~100%，GPU Insertion和Zip-tie cutting也接近成功）。
- 多机器人并行显著加速策略改进：从1台到8台，Push-T时间从~5h降至~2h，Pin Insertion从>1.5h降至~40min。
- 跨任务可迁移：从Pin Insertion积累的经验（如BC正则化、参数调优）可指导GPU Insertion学习。
- 自动发现VLA与程序代码结合策略：在RoboCasa和真实世界中，智能体学会先悬停再抓取，提升成功率。
- 资源利用率：机器人利用率随规模增加而下降（因为智能体更多时间用于日志分析、代码合并），令牌消耗超线性增长，表明效率与规模的权衡。

## 7. 优点
- **首次提出并实现物理自主研究（physical autoresearch）**：将编码智能体闭环部署到真实机器人，减少人工监督。
- **模块化设计**：环境构建与策略改进分离，降低风险，API可复用。
- **灵活的方法支持**：适用范围从启发式到深度学习，甚至结合VLA，适应性极强。
- **多智能体协作机制**：基于Git实现去中心化协作，易于扩展至大规模集群，且能自动分享知识。
- **提出新的效率指标**（MRU、MTU、Token-to-Success），为评估物理自主研究的资源效率提供标准。
- **跨任务迁移**：展示了知识总结与复用，减少重复劳动。

## 8. 不足与局限
- **资源利用率低**：机器人利用率在8智能体时降至约50%，GPU利用率更低（约25%），智能体大量时间用于非操作任务（调试、等待LLM推理）。
- **令牌成本超线性增长**：增加智能体数量虽加快速度，但总令牌消耗增长远快于理想线性，经济性差。
- **环境构建仍需一次人工投入**：虽然声称一次性的，但实际中仍需人类提供任务描述、安全约束、种子示范等，并非完全零人工。
- **实验规模有限**：仅使用8台同构机器人，未测试更大规模或异质机器人集群；仅涵盖4种任务，未评估高度动态或长时域任务。
- **无统计学显著性分析**：未报告方差或置信区间，对于真实世界非确定性环境，单一成功率可能不够鲁棒。
- **对感知模块依赖强**：如自动验证依赖SAM3等工具，在小物体或模糊场景下可能失败（论文也承认此点）。
